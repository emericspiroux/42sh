-------------- Introduction ---------------------
 42sh is a command interpreter which is a
 replica of the unix shell with the functionality
 (except the interpretation of script). You can
 use it instead of your shell by changing your
 default interpreter was using the chsh [LOGIN]
 then change the line "shell" in the root of
 your folder containing your 42sh.
 Thank you use the 42sh.

Ronan, Franklin, Emeric.

-------------- Install --------------------------

 1 - Make

-------------- Uninstall ------------------------

 2 - Make fclean
 	erase 42sh and .o files
 or
 3 - Make clean
 	Erase .o files.

 -------------- Execution -----------------------

 4 - ./42sh

 -------------- Exit ----------------------------

 5 - 42sh?> exit
